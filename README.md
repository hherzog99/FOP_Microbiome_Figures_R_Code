# R Code for the FOP Microbiome Paper Stats and Figures
This repository contains the R code for the FOP microbiome paper figures.

## Folders

fop_mouse_microbiome_functional_analysis/

Contains R scripts for downstream analysis of functional shotgun metagenomic sequencing data.
See the README within this folder for additional details.

fop_mouse_taxonomic_analysis/

Contains R scripts for downstream taxonomic analysis of shotgun metagenomic sequencing data.
See the README within this folder for additional details.

mmp9_script/

Contains the R script used to generate Figure 3A.
See the README within this folder for additional details.

scripts/

Contains the R project and scripts used to generate Figures 2D, 2F, 3B, and 4B.
See the README within this folder for additional details.
